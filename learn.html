Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML menggambarkan struktur halaman web dengan menggunakan berbagai elemen dan tag. Setiap elemen HTML memiliki fungsi tertentu, seperti menampilkan teks, gambar, video, dan berbagai konten lainnya.

Sejarah HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada akhir 1980-an dan awal 1990-an. HTML telah mengalami banyak perubahan dan pembaruan sejak saat itu. Versi terbaru dari HTML adalah HTML5, yang diperkenalkan pada tahun 2014 dengan banyak fitur baru yang mendukung multimedia, grafis, dan berbagai fungsionalitas modern lainnya.

Pembelajaran Dasar HTML

1. Struktur Dasar HTML

Setiap dokumen HTML dimulai dengan deklarasi <!DOCTYPE html> yang menunjukkan versi HTML yang digunakan. Berikut adalah contoh struktur dasar dokumen HTML:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Belajar HTML</title>
</head>
<body>
    <!-- Konten halaman web Anda akan diletakkan di sini -->
</body>
</html>

2. Elemen Heading

Heading digunakan untuk membuat judul dan subjudul dalam dokumen HTML. Ada enam tingkat heading, dari <h1> hingga <h6>, dengan <h1> sebagai yang terbesar dan <h6> sebagai yang terkecil.

<h1>Judul Utama</h1>
<h2>Subjudul</h2>
<h3>Sub-subjudul</h3>

3. Elemen Paragraf

Paragraf digunakan untuk menampilkan teks dalam bentuk blok. Elemen ini dilambangkan dengan tag <p>.

<p>Ini adalah paragraf pertama dalam HTML.</p>
<p>Ini adalah paragraf kedua.</p>

4. Elemen Tautan

Tautan digunakan untuk membuat hyperlink ke halaman lain atau sumber daya lain. Elemen ini dilambangkan dengan tag <a>.

<a href="https://www.example.com">Ini adalah tautan</a>

5. Elemen Gambar

Gambar digunakan untuk menampilkan gambar dalam dokumen HTML. Elemen ini dilambangkan dengan tag <img> dan menggunakan atribut src untuk menentukan sumber gambar.

<img src="image.jpg" alt="Contoh Gambar">

6. Elemen Daftar

HTML menyediakan dua jenis daftar: ordered list (daftar berurutan) dan unordered list (daftar tidak berurutan). Ordered list menggunakan tag <ol> dan unordered list menggunakan tag <ul>.

<h3>Ordered List</h3>
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

<h3>Unordered List</h3>
<ul>
    <li>Item A</li>
    <li>Item B</li>
    <li>Item C</li>
</ul>

7. Elemen Tabel

Tabel digunakan untuk menampilkan data dalam bentuk grid. Elemen ini dilambangkan dengan tag <table>, dengan baris tabel menggunakan tag <tr> dan sel menggunakan tag <td> untuk data dan <th> untuk header.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

8. Elemen Form

Formulir digunakan untuk mengumpulkan input pengguna. Elemen ini dilambangkan dengan tag <form> dan menggunakan berbagai elemen input seperti <input>, <textarea>, dan <button>.

<form action="/submit" method="post">
    <label for="name">Nama:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Contoh Kode HTML Lengkap

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Belajar HTML</title>
</head>
<body>
    <h1>Judul Utama</h1>
    <p>Ini adalah paragraf pertama dalam HTML.</p>
    <a href="https://www.example.com">Ini adalah tautan</a>
    <img src="image.jpg" alt="Contoh Gambar">
</body>
</html>
